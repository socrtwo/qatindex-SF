***Microsoft Office QAT Index***
The QAT is the Quick Access Toolbar on the top or bottom of the MS Office 2007 and 2010 command ribbon or menu.  it allows access to the 1000-1500 commands available but possibl7y hudeen (not of the ribbons).  THe se commands include most of the commands from earlier versions of the programs.

These tywo excel files are  the beginnings of indexes for the Excel and PowerPoint QAT or Quick Access Toolbar. The VBA used and included (I think I included it) allows the extraction of the name, tooltip, SuperTooltip (descriptiopn where available) icon and maybe filled URL calls to the Microsoft and YouTube reseources about the each command if they exist.

These two files could be used a basis for a web index for the QAT.  

There are a few spreadsheets available for free from Microsoft and one or two other places, that provide similar information.  The idea here was tro combine all the spreadsheet info in one location and open a website with the index displayed in some easily usable form.
