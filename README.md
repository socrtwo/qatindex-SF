# Microsoft Office QAT Index

An index for the Quick Access Toolbar (QAT) in Microsoft Office 2007/2010, covering Excel and PowerPoint commands. Includes VBA code usable with Word.

**Language:** VBA / Excel

## Features

- Searchable index of all QAT commands
- Covers Excel and PowerPoint (Office 2007/2010)
- VBA code included for Word integration
- Helps discover hidden toolbar commands
